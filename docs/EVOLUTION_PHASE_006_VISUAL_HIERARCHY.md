# Fase 006 — hierarquia visual

A navegação agora separa com maior clareza três níveis: grupos, destinos disponíveis e tela ativa.

O estado ativo não depende somente de cor. Ele combina peso tipográfico, trilho lateral, superfície do ícone e o `aria-current` já existente. Os itens inativos permanecem legíveis, porém visualmente secundários.

Nenhuma rota, permissão ou ação foi alterada.
