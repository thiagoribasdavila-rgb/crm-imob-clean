# Fase 71 — Contrato canônico dos dados da lead

O contrato V1 normaliza e-mail, telefone, origem e identidade; mede dez dimensões que somam 100 pontos; registra lacunas e separa explicitamente qualidade de dados de probabilidade de compra.

O cálculo é local, determinístico e sem custo de IA. A API respeita a visibilidade hierárquica e não retorna telefone, e-mail, notas ou documentos no painel agregado.
