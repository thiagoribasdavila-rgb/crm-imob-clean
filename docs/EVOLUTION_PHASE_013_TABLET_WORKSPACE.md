# Fase 013 — workspace para tablet

Entre 768 e 1023 px, o Atlas agora usa margens de 24 px e reserva espaço adequado para o dock inferior. O dock fica centralizado e limitado a 560 px, deixando de cobrir visualmente toda a base da interface.

A barra superior oculta apenas a cópia textual do usuário, mantendo criação de lead, busca global, notificações, perfil e saída. O menu lateral aberto ganha largura confortável para toque sem dominar a tela.

As regras de desktop e celular permanecem isoladas.
