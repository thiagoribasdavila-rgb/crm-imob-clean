# Fase 70 — Homologação completa do bloco de projetos

Cada empreendimento precisa passar por oito gates: incorporadora canônica, cadastro/localização, catálogo, estoque, versão comercial, materiais essenciais, estudo regional e dossiê da IA.

O aceite é exclusivo da diretoria. O servidor revalida os dados no momento da decisão, preserva um snapshot e não autoriza produção automaticamente.

## Homologação real

Aplicar a migration, abrir `/developments/homologation`, comprovar bloqueios para cada gate, completar um projeto, registrar o aceite e repetir após alterar/vencer uma evidência. Validar diretoria, demais perfis e dois tenants.
